## About LGZ bot
* 这是[Mirai](https://github.com/mamoe/mirai)框架的插件,依赖于[Mirai Console](https://github.com/mamoe/mirai-console)运行.
* 本插件可以为使用者提供群消息合规性检查,并作出相应的处罚.
* 还有一些有趣的小东西 \>_\<
